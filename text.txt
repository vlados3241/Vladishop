СТРАНИЦЫ С ТОВАРАМИ 
Для всех категорий страница с товарами будет одна, но с товарами по выбранной категории.
Либо 4 картинки товара в ряду на белом фоне без окантовки, либо две картинки с товаром по размерам как 4. Карточки товара будут находиться рядом  
Всего будет +-12 товаров в каждой категории.
Под каждой карточкой товара будет Название, категория, цена, скидки(если есть) и допустимые размеры. 
В станицах данных категорий будет список брендов

Личный кабинет 
Экран разделен на две части, с одной строный "Вход", а  другой - "Регистрация"
Эти части будут разделены едва заметной вертикальной линией
Если пользователь авторизован, то в личном кабинете должна быть инфрмация о пользователе; имя, фамилия, возраст, электронная почта, доступные спосмобы оплаты
 
Вход 
1. Место для ввода почты
2. Место для ввода пароля 
3. Надпись-ссылка - забыл пароль или восстановление пароля 
4. Кнопка Войти в цветах сайта 

Регистрация 
1. Перечень того, что нужно для регистрации
2. Кнопка Регистрация в цветах сайта 
При нажатии кнопки Регистрация открывется новая страница 
Там поля ввода номера телефона или электронной почты. 
Кнопка Далее. При нажатии кнопки Далее открывается новая страница с полем для ввода код-пароля
Поле ввода код-пароля, который пришел на телефон или почту 
Кнопка Далее. При нажатии кнопки Далее открывается новая страница с полем для ввода пароля
Поле для ввода нового пароля и кнопка Создать аккаунт
При нажатии кнопки Создать аккаунт открывается страница с авторизованным пользователем

При нахождении вещи через поисковую строку на станице отображается количество найденных вещей (432 items)
